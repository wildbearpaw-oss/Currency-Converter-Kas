
# Currency Converter — Конвертер валют

## Описание программы

**Currency Converter** — это GUI‑приложение для конвертации валют с использованием актуальных курсов через внешний API. Приложение позволяет выбирать валюты, вводить сумму для конвертации, выполнять расчёт и просматривать историю операций. Данные истории сохраняются в формате JSON и могут быть загружены при следующем запуске.

### Возможности:
- ✅ Выбор валюты «из» и «в» из выпадающих списков
- ✅ Ввод суммы для конвертации
- ✅ Кнопка запуска конвертации
- ✅ Отображение результата конвертации
- ✅ Сохранение истории операций в JSON‑файл
- ✅ Загрузка истории операций из JSON‑файла при запуске
- ✅ Валидация вводимых данных (проверка на положительное число)
- ✅ Использование внешнего API для получения актуальных курсов валют

## Примеры использования

### Конвертация валюты

1. Выберите валюту «из» в первом выпадающем списке (например, `USD`).
2. Выберите валюту «в» во втором выпадающем списке (например, `RUB`).
3. Введите сумму для конвертации в поле «Сумма» (например, `100`).
4. Нажмите кнопку **«Конвертировать»**.
5. Результат отобразится под полем ввода, а операция добавится в таблицу истории.

### Просмотр истории

- Все выполненные операции автоматически сохраняются и отображаются в таблице **«История конвертаций»**.
- При перезапуске приложения история загружается из файла `history.json`.

### Тестовые данные (примеры для проверки)

Попробуйте выполнить следующие конвертации для тестирования:

| Из валюты | В валюту | Сумма | Ожидаемый результат (примерный) |
|----------|---------|-------|-------------------------------|
| USD      | RUB     | 100   | ~9 000–10 000 RUB           |
| EUR      | USD     | 50    | ~54–55 USD                  |
| GBP      | EUR     | 200   | ~235–240 EUR                |
| JPY      | CNY     | 1 000 | ~6–7 CNY                    |

**После конвертации:**

1. Убедитесь, что результат отображается корректно.
2. Проверьте, что операция появилась в таблице истории.
3. Закройте приложение, запустите снова — история должна загрузиться автоматически.

## Валидация данных

Приложение проверяет:

- Сумма должна быть положительным числом (больше нуля).
- Поля выбора валют не должны быть пустыми.
- При некорректном вводе выводится сообщение об ошибке.

## Получение API‑ключа

Для работы приложения требуется API‑ключ от сервиса обмена валют.

**Как получить API‑ключ (на примере exchangerate-api.com):**

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com).
2. Зарегистрируйтесь (создайте аккаунт).
3. Получите бесплатный API‑ключ в личном кабинете.
4. Создайте файл `.env` в корне проекта и добавьте ключ:

   ```env
   API_KEY=ваш_api_ключ_здесь
   
Приложение автоматически подгрузит ключ из этого файла.

Важно: Не коммитьте файл .env в репозиторий. Добавьте его в .gitignore.
